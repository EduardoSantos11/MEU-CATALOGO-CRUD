# Catálogo de Filmes e Séries: Aplicação CRUD com React e JSON Server

Este repositório contém o projeto final para a disciplina de Frameworks Front-End. O projeto implementa um sistema completo para gerenciamento de um catálogo pessoal de filmes e séries, desenvolvido do zero para demonstrar a implementação de operações CRUD (Create, Read, Update, Delete) em um ambiente Front-End moderno.

## Objetivo Geral

O objetivo deste projeto foi desenvolver uma aplicação web funcional, utilizando Git, GitHub, React, TypeScript e JSON Server, implementando todas as operações de gerenciamento de dados com persistência no arquivo `db.json`.

### Funcionalidades Implementadas

O sistema permite as seguintes operações, essenciais para o escopo do projeto:

* **Listagem (Read):** Exibição de todos os registros da coleção.
* **Cadastro (Create):** Adição de novos itens usando requisições POST.
* **Edição (Update):** Modificação de registros existentes com requisições PUT/PATCH.
* **Exclusão (Delete):** Remoção de itens específicos.
* **Persistência de Dados:** Todos os dados são salvos no arquivo `db.json` usando JSON Server.

## Tecnologias e Componentização

A aplicação foi inicializada com **Vite + React + TypeScript** e utiliza a seguinte stack técnica:

| Categoria | Tecnologia | Uso no Projeto |
| :--- | :--- | :--- |
| **Frontend** | React com TypeScript | Desenvolvimento da interface e Componentização. |
| **Backend Simulado** | JSON Server | Simulação de API REST para persistência local. |
| **Estilização** | CSS Puro | Layout e usabilidade (IHC). |
| **Comunicação** | Fetch API | Realização das requisições CRUD (GET, POST, PUT, DELETE). |
| **Versionamento** | Git e GitHub | Controle de versão do código e publicação do repositório. |
