# Catálogo de Filmes e Séries: Aplicação CRUD com React e JSON Server

Este repositório contém o projeto final para a disciplina de Frameworks Front-End. O projeto implementa um sistema completo para gerenciamento de um catálogo pessoal de filmes e séries, desenvolvido do zero para demonstrar a implementação de operações CRUD (Create, Read, Update, Delete) em um ambiente Front-End moderno.

##  Objetivo e Escopo do Projeto

[cite_start]O objetivo geral deste projeto foi desenvolver uma aplicação web funcional, evoluindo-a do zero até um sistema completo, utilizando as tecnologias listadas para implementar todas as operações de gerenciamento de dados[cite: 4, 5].

### Funcionalidades Implementadas (CRUD)

O sistema permite as seguintes operações, essenciais para o escopo do projeto:

* [cite_start]**Listagem (Read):** Exibição de todos os registros da coleção[cite: 12].
* [cite_start]**Cadastro (Create):** Adição de novos itens usando requisições POST[cite: 13].
* [cite_start]**Edição (Update):** Modificação de registros existentes com requisições PUT/PATCH[cite: 14].
* [cite_start]**Exclusão (Delete):** Remoção de itens específicos[cite: 15].
* [cite_start]**Persistência de Dados:** Todos os dados são salvos no arquivo `db.json` usando JSON Server[cite: 16].
